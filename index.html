> Pheatthph

Pheatthph

A game with a cool aesthetic.
Pheatthph is a game some friends and I cooked up during one of our classic brainstorming sessions.

Pheatthph was made in UNITY and the project is currently still ongoing, with no planned end date.
The main idea behind the game is that you travel through different dimentions by going into this creature called pheatthph, a massive fetus in the sky.

As you might be able to tell the concept is really weird but it's surprisingly fun and gives a good aesthetic. The main aesthetic is also that we went for PS1 style graphics as you might be able to see in the footage above

> Goldfish Dreams

Goldfish Dreams

Our first game jam
This is a hotline miami inspired game where a Goldfish has to kill everyone from "Goldfish Killing Corp.". However, the fish obviously can't hold any guns, much less fire them, so the fish instead switches from person to person in order to accomplish its task.

This game was made for a game jam we had at school, which lasted around about 48 hours total.
I mainly did all the asset art whilst my friend Kick did the level art and music and Faelin did most of the coding.

I love this project because it's the first project I've actually managed to finish with my friends, and I also developed my art skills but above all else my ability to work on a game with a team.
Here you can see some of the assets I made for the game.
