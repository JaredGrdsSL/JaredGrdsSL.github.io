> Dreamcatcher

Dreamcatcher

Passion project.
Dreamcatcher is a game I made with an idea that just spawned in my head, namely the 360 FOV. It is not finished yet, however I do plan on finishing it eventually
As you can see the visuals are very trippy and do take a bit to get used to, however once you get used to them, you can actually use it practically.
This is the pause screen, another random idea that worked really well in game. I call it an immersive experience, others call it the most intrusive pause screen they've ever seen, to each their own I guess.

What I've learned

This game taught me most of what I know about unity 3D now, including a bit about ShaderGraph, ProBuilder, and World Space UI, among others. It has also taught me some important lessons in how to come up with ideas for projects and how to actually turn them into a game instead of just a tech-demo. This is something I immediately already applied to my next project, Qorst.

> Goldfish Dreams

Goldfish Dreams

Our first game jam
This is a hotline miami inspired game where a Goldfish has to kill everyone from "Goldfish Killing Corp.". However, the fish obviously can't hold any guns, much less fire them, so the fish instead switches from person to person in order to accomplish its task.

This game was made for a game jam we had at school, which lasted around about 48 hours total.
I mainly did all the asset art whilst my friend Kick did the level art and music and Faelin did most of the coding.
Here you can see some of the assets I made for the game. I bought Aseprite just for this project however I continue to use it as it is just that good.

What I've learned

I love this project because it's the first project I've actually managed to finish with my friends, and I also developed my art skills but above all else my ability to work on a game with a team. It's also the first game jam I ever took part of, and it was fun to see how far we could get in a short amount of time. We even continued progress on this game after the jam to fix some bugs and add some features, and the game is out on Itch.io!

> Pheatthph

Pheatthph

A game with a cool aesthetic.
Pheatthph is a game some friends and I cooked up during one of our classic brainstorming sessions.

Pheatthph was made in UNITY and the project is currently still ongoing, with no planned end date.
The main idea behind the game is that you travel through different dimensions by going into this creature called pheatthph, a massive fetus in the sky.

As you might be able to tell the concept is really weird, but it's surprisingly fun and gives a good aesthetic. The main aesthetic is also that we went for PS1 style graphics as you might be able to see in the footage above

What I've learned

This game was my first time using GitHub with others. We first wanted to use PlasticSCM but that was too difficult to set up properly and I think we also had to pay, so instead we decided to learn and use GitHub, which has become one of the handiest things I ever learnt programming wise. What you see pictured above is also my first time using ProBuilder in Unity, which has also come in very handy.

All in all this game has just been very helpful in learning even if it's not much of a game right now.
